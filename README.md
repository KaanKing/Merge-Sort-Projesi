# Merge-Sort-Projesi

[16,21,11,8,12,22] -> Merge Sort


  1. Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
    
    CEVAP;
    
            [16,21,11,8,12,22]
                     / \
            [16,21,11] [8,12,22]
                 / \        / \
            [16][21,11] [8][12,22]
             |    / \    |   / \
           [16][21][11] [8][12][22]
              |   \ /     |   \ /
            [16][11,21] [8][12,22]
                \  /       \  /
            [11,16,21][8,12,22]
                     \  /
             [8,11,12,16,21,22]
